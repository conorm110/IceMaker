start=init on activation will run checks on dependencies to make sure they
are installed properly. if they are, dependencies_installed=true. if not
dependencies_installed=false. if start=init, dependencies_installed will be
read as false no matter what. python will be set on activation if start=init
and defaults to python on mac and linux. if on windows, python=py but if py 
fails to execute python=python will be tested

START
start=init
dependencies_installed=false
python=#
END
